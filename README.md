# Wordaholics

Wordaholics is a **Wordle-inspired word guessing game** built using **React** and **Vite**. The game challenges players to guess a hidden word within a limited number of attempts while providing visual feedback for correct and misplaced letters.

## Live Demo

Play the game here:
https://unrivaled-starlight-3d168a.netlify.app/

## Features

* Wordle-style gameplay
* Interactive feedback for correct and incorrect letters
* Support for multiple word lengths (4, 5, and 6 letter words)
* Clean and responsive user interface
* Component-based architecture using React
* 9,000+ built-in words dictionary

## Tech Stack

* React
* Vite
* JavaScript
* HTML
* CSS

## Run Locally

Clone the repository:

```
git clone https://github.com/Shylad2007/wordaholics.git
```

Move into the project folder:

```
cd wordaholics
```

Install dependencies:

```
npm install
```

Start the development server:

```
npm run dev
```

The application will run at:

```
http://localhost:5173
```
